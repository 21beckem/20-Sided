# Dragon's Quill
Your next D&D Teammate

## Project Description

Dragon's Quill is a Dungeons & Dragons focused web application centered around virtual tabletop functionality. It provides a comprehensive platform for D&D players and Dungeon Masters to create, manage, and share interactive maps for their gaming sessions.

### Core Features

- **User Accounts**: Create and manage your own account
- **Map Builder**: Design custom maps for your D&D sessions
- **Map Collection**: Save and organize your created maps
- **Map Chunks**: Save reusable map segments for quick building
- **Public Sharing**: Share your maps with the community
- **Play Sessions**: Start interactive sessions with shareable links for players
- **Real-time Collaboration**: Players can view maps during gameplay through shared session links

## Technical Stack

### Frontend
- HTML, CSS, Vanilla JavaScript
- Optional: Svelte for reusable UI components
- Responsive design with DOM manipulation
- Form-based user input flows

### Backend
- Node.js with Express
- RESTful API architecture
- MongoDB for data persistence
- Error handling and input validation

### Integration
- HTTP communication via Fetch API
- Full CRUD operations
- Session management
